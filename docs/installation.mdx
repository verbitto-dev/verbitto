## Prerequisites

You need the following tools installed:

- **Rust** (latest stable) — via [rustup](https://rustup.rs/)
- **Solana CLI** ≥ 1.18 — via [Solana Install](https://docs.solana.com/cli/install-solana-cli-tools)
- **Anchor CLI** 0.30.1 — via `cargo install --git https://github.com/coral-xyz/anchor avm && avm install 0.30.1 && avm use 0.30.1`
- **Node.js** ≥ 18 and **pnpm** ≥ 9

## Clone the Repository

```bash
git clone https://github.com/your-org/verbitto.git
cd verbitto
```

## Install Dependencies

```bash
pnpm install
```

## Build the Program

```bash
anchor build
```

This compiles the Rust program and generates:
- `target/deploy/task_escrow.so` — the compiled BPF program
- `target/idl/task_escrow.json` — the Anchor IDL
- `target/types/task_escrow.ts` — TypeScript types

## Configure Solana

```bash
solana config set --url devnet
solana-keygen new --outfile ~/.config/solana/devnet.json
solana config set --keypair ~/.config/solana/devnet.json
solana airdrop 2
```

## Verify the Program ID

```bash
solana address -k target/deploy/task_escrow-keypair.json
```

Ensure this matches `declare_id!` in `programs/task-escrow/src/lib.rs` and the `[programs.devnet]` section in `Anchor.toml`.

## Frontend Setup

```bash
pnpm dev
```

The development server starts at `http://localhost:3000`. The frontend auto-connects to Solana devnet via the wallet adapter.
