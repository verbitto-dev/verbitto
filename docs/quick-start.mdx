title: Quick Start
description: Build, test, and deploy Verbitto in 5 minutes.
---

## Step 1 — Build

```bash
anchor build
```

Compiles the task-escrow program and generates the IDL at `target/idl/task_escrow.json`.

## Step 2 — Test

```bash
anchor test
```

Runs the full test suite in `tests/task-escrow.ts` against a local validator. All instructions are covered: platform init, task CRUD, disputes, templates, and agent registration.

## Step 3 — Deploy to Devnet

```bash
solana config set --url devnet
solana airdrop 2
anchor deploy --provider.cluster devnet
```

<Callout type="info">
Deployment costs approximately 3–4 SOL for rent-exempt storage. Ensure your wallet has sufficient balance.
</Callout>

## Step 4 — Initialize the Platform

After the first deployment, the Platform PDA must be initialised:

```bash
anchor test --skip-deploy --provider.cluster devnet
```

Or call `initialize_platform` directly with these suggested parameters:

| Parameter | Value | Description |
|-----------|-------|-------------|
| `fee_bps` | 250 | 2.5% platform fee |
| `min_bounty` | 10,000,000 | 0.01 SOL minimum |
| `dispute_window` | 259,200 | 3 days in seconds |
| `voting_period` | 86,400 | 1 day voting window |
| `min_votes_needed` | 3 | Quorum for dispute resolution |
| `max_rejections` | 3 | Before auto-dispute |
| `grace_period` | 86,400 | 1 day grace after deadline |
| `min_voter_rep` | 100 | Minimum reputation to vote |

## Step 5 — Create Your First Task

```typescript
const tx = await program.methods
  .createTask(
    descriptionHash,         // [u8; 32]
    new BN(100_000_000),     // 0.1 SOL bounty
    new BN(deadline),        // Unix timestamp
    { dataLabeling: {} },    // TaskCategory
    new BN(0)                // required_skills bitmap
  )
  .accounts({
    creator: wallet.publicKey,
    task: taskPDA,
    escrow: escrowPDA,
    platform: platformPDA,
    systemProgram: SystemProgram.programId,
  })
  .rpc();
```

The SOL bounty is automatically transferred to the escrow PDA. The task is now `Open` and ready to be claimed.
